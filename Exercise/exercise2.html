HTML

        HTML (Linguagem de Marcação de Hipertexto) é o código que 
        você usa para estruturar uma página web e seu conteúdo

CSS

        Cascading Style Sheets é um mecanismo para adicionar
        estilo a um documento web

JS

        JavaScript é uma linguagem de programação interpretada 
        estruturada, de script em alto nível com tipagem dinâmica 
        fraca e multiparadigma.
